# شروع سریع - OpenAI (5 دقیقه)

با مدل‌های GPT OpenAI اجرا می‌شود. سریع، قدرتمند و ساده.

## پیش نیاز

1. **Docker Desktop** نصب شده است
   - [از اینجا دانلود کنید](https://www.docker.com/products/docker-desktop/)
   - از قبل دارید؟ به مرحله 2 بروید

2. **کلید OpenAI API** (الزامی)
   - به https://platform.openai.com/api-keys بروید
   - ایجاد حساب → ایجاد کلید مخفی جدید
   - حداقل 5 دلار اعتبار به حساب خود اضافه کنید
   - کلید را کپی کنید (با 'sk-' شروع می شود)

## مرحله 1: ایجاد پیکربندی (1 دقیقه)

یک پوشه جدید «open-notebook» ایجاد کنید و این فایل را اضافه کنید:

**docker-compose.yml**:
`` یامل
خدمات:
  surrealdb:
    تصویر: surrealdb/surrealdb:v2
    دستور: start --user root --pass password --bind 0.0.0.0:8000 rocksdb:/mydata/mydatabase.db
    پورت ها:
      - "8000:8000"
    حجم ها:
      - ./سوررئال_داده:/mydata

  open_notebook:
    تصویر: lfnovo/open_notebook:v1-latest
    pull_policy: همیشه
    پورت ها:
      - "8502:8502" # رابط کاربری وب
      - "5055:5055" # API
    محیط زیست:
      # کلید رمزگذاری برای ذخیره اعتبار (الزامی)
      - OPEN_NOTEBOOK_ENCRYPTION_KEY=تغییر-me-to-a-secret-string

      # پایگاه داده (الزامی)
      - SURREAL_URL=ws://surrealdb:8000/rpc
      - SURREAL_USER = ریشه
      - SURREAL_PASSWORD=رمز عبور
      - SURREAL_NAMESPACE=open_notebook
      - SURREAL_DATABASE=نوت بوک باز
    حجم ها:
      - ./notebook_data:/app/data
    بستگی_به:
      - surrealdb
    راه اندازی مجدد: همیشه

```

**ویرایش فایل:**
- «رشته تغییر من به یک مخفی» را با راز خود جایگزین کنید (هر رشته ای کار می کند)

## مرحله 2: شروع خدمات (1 دقیقه)

ترمینال را در پوشه 'open-notebook' خود باز کنید:

``باش
```

## مرحله 3: دسترسی به نوت بوک باز (فوری)

مرورگر خود را باز کنید:
```
http://localhost:8502
```

باید رابط Open Notebook را ببینید!

## مرحله 4: ارائه دهنده OpenAI خود را پیکربندی کنید (1 دقیقه)

1. به **تنظیمات** → **کلیدهای API** بروید
2. روی **افزودن اعتبار** کلیک کنید
3. ارائه دهنده را انتخاب کنید: **OpenAI**
4. نامی به آن بدهید (به عنوان مثال، «کلید OpenAI من»)
5. کلید OpenAI API خود را جایگذاری کنید
6. روی **ذخیره** کلیک کنید
7. روی **Test Connection** کلیک کنید - باید موفقیت را نشان دهد
8. روی **Discover Models** → **Register Models** کلیک کنید

مدل‌های OpenAI شما اکنون در دسترس هستند!

---

## مرحله 5: اولین نوت بوک خود را بسازید (1 دقیقه)

1. روی **New Notebook** کلیک کنید
2. نام: "تحقیق من"
3. روی **ایجاد** کلیک کنید

---

## مرحله 6: اضافه کردن یک منبع (1 دقیقه)

1. روی **افزودن منبع** کلیک کنید
2. **پیوند وب** را انتخاب کنید3. چسباندن: «https://en.wikipedia.org/wiki/Artificial_intelligence»
4. روی **افزودن** کلیک کنید
5. برای پردازش صبر کنید (30-60 ثانیه)

---

## مرحله 7: با محتوای خود گپ بزنید (1 دقیقه)

1. به **چت** بروید
2. نوع: "هوش مصنوعی چیست؟"
3. روی **ارسال** کلیک کنید
4. تماشا کنید که GPT با اطلاعات منبع شما پاسخ می دهد!

## عیب یابی

### "پورت 8502 در حال استفاده است"

پورت را در docker-compose.yml تغییر دهید:
`` یامل
پورت ها:
  - "8503:8502" # به جای آن از 8503 استفاده کنید
```

سپس به «http://localhost:8503» دسترسی پیدا کنید

### "کلید API کار نمی کند"

1. به **تنظیمات** → **کلیدهای API** بروید
2. روی **Test Connection** روی اعتبار OpenAI خود کلیک کنید
3. اگر ناموفق بود، کلید خود را در https://platform.openai.com تأیید کنید
4. اعتبارنامه را حذف کنید و یک مورد جدید با کلید صحیح ایجاد کنید

### "نمی توان به سرور متصل شد"

``باش
docker ps # همه سرویس‌های در حال اجرا را بررسی کنید
docker compose logs # مشاهده گزارش‌ها
docker compose راه اندازی مجدد # همه چیز را راه اندازی مجدد کنید
```
