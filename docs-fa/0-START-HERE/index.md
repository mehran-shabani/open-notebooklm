## چه کاری می توانید انجام دهید؟

- 📄 **آپلود محتوا**: پی دی اف، لینک های وب، صدا، ویدئو، متن
- 🤖 **چت با هوش مصنوعی**: در مورد اسناد خود با نقل قول سوال بپرسید
- 📝 **ایجاد یادداشت**: هوش مصنوعی خلاصه و بینش ایجاد می کند
- 🎙️ **ایجاد پادکست**: تحقیق را به محتوای صوتی حرفه ای تبدیل کنید
- 🔍 **جستجو **: جستجوی تمام متن و معنایی در تمام محتوا
- ⚙️ **تبدیل **: استخراج بینش، تجزیه و تحلیل مضامین، ایجاد خلاصه

## چرا نوت بوک را باز کنید؟

| ویژگی | باز کردن نوت بوک | نوت بوک LM |
|---------|---|---|
| **حریم خصوصی** | میزبانی خود، کنترل شما | ابر، سرورهای گوگل |
| **انتخاب هوش مصنوعی** | 17+ ارائه دهندگان | فقط مدل های گوگل |
| **سخنران پادکست** | 1-4 قابل تنظیم | 2 فقط |
| **هزینه** | کاملا رایگان | رایگان (اما داده های شما) |
| **آفلاین** | بله | نه |

## پیش نیاز

- **Docker**: همه مسیرها از Docker (رایگان) استفاده می کنند
- **ارائه دهنده هوش مصنوعی**: یا یک کلید API ابری یا استفاده از مدل های محلی رایگان (Ollama)
