# واژهنامه فارسی محصول

این واژهنامه برای یکدست‌سازی متن رابط کاربری فارسی (fa-IR) تهیه شده است.

## اصطلاحات اصلی

| انگلیسی | واژه قانون فارسی | توضیح کاربرد |
|---|---|---|
| منبع | منبع | هر فایل/لینک/متن وارد شده به سیستم |
| دفترچه یادداشت | دفتر | فضای کاری شامل منابع و یادداشت ها |
| توجه | یادداشت | متن کاربر داخل دفتر |
| بینش | بینش | خلاصه/برداشت خودکار از محتوا |
| بپرسید | پرسش | حالت پرسید از پایگاه دانش |
| جستجو | جستجو | جستجوی حالت مستقیم در محتوا |
| تحول | تبدیل | تبدیل محتوا با پرامپت یا قالب |
| مدل | مدل | مدل هوش مصنوعی |
| تعبیه | امبدینگ | نمایش برداری متن برای بازیابی |
| زمینه | زمینه | ارسالی همراه درخواست |
| اعتبارنامه | اعتبارنامه | اطلاعات ارتباط با ارائه مدل |
| کلید API | کلید API | کلید دسترسی به سرویس |
| حذف | حذف | حذف مورد انتخابی |
| ذخیره | ذخیره شده | ثبت تغییرات |
| تلاش مجدد | جستجو مجدد | اجرای مجدد عملیات شکست خورده |

##اصول لحن

- لحن: محترمانه، ساده و مستقیم.
- CTAها: فعلی و روشن باشند (مثل «ذخیره تغییرات»، «حذف دائمی»).
- پیام خطا: کوتاه، روشن و قابل اقدام.
- پیام‌های دسترسی‌پذیری (aria/alt/title): توصیفی و کاربرمحور باشند.

## الگوهای پیشنهادی

- خطای عمومی: «خطای غیرمنتظره‌ای رخ داد»
- خطای بارگذاری: «بارگذاری {مورد} ناموفق بود»
- موفقیت: «{مورد} با موفقیت ذخیره شد»
- حذف: «آیا از حذف این {مورد} مطمئن هستید؟»
