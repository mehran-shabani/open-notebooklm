## 🚀 اولویت های توسعه فعلی

ما فعالانه به دنبال کمک هستیم:

1. **افزایش فرانتند** - بهبود رابط کاربری Next.js/React با به‌روزرسانی‌های بی‌درنگ
2. ** عملکرد ** - بهینه سازی های پردازش و ذخیره سازی غیر همگام
3. **تست ** - پوشش تست را در بین اجزاء گسترش دهید
4. **مستندات** - نمونه های API و راهنمای توسعه دهندگان
5. **ادغام** - منابع محتوای جدید و ارائه دهندگان هوش مصنوعی

به مشکلات GitHub با برچسب «نسخه اول خوب» یا «کمک مورد نیاز» مراجعه کنید.
