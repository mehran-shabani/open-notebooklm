## سیستم مورد نیاز

### حداقل
- **رم **: 4 گیگابایت
- ** فضای ذخیره سازی **: 2 گیگابایت برای برنامه + فضای برای اسناد
- **CPU **: هر پردازنده مدرن
- **شبکه**: اینترنت (اختیاری برای راه اندازی آفلاین)

### توصیه می شود
- **رم **: 8 گیگابایت +
- ** فضای ذخیره سازی **: 10 گیگابایت + برای اسناد و مدل ها
- **CPU**: پردازنده چند هسته ای
- **GPU**: اختیاری (مدل های هوش مصنوعی محلی را سرعت می بخشد)

### مبتنی بر فضای ابری (پرداخت در صورت تمایل)
- **OpenAI** - GPT-4، GPT-4o، سریع و توانا

**هزینه**: معمولاً 0.01 تا 0.10 دلار در هر هزار توکن
**سرعت**: سریع (زیر ثانیه)
**حریم خصوصی**: داده های شما به ابر ارسال می شود

### محلی (رایگان، خصوصی)
- **Ollama** - مدل های منبع باز را به صورت محلی اجرا کنید
- ** LM Studio ** - برنامه دسکتاپ برای مدل های محلی
- **مدل های صورت در آغوش گرفته** - دانلود و اجرا کنید

**هزینه**: 0 دلار (فقط برق)
**سرعت**: بستگی به سخت افزار شما دارد (آهسته تا متوسط)

- [ ] **کلید API ارائه دهنده AI ** (OpenAI، Anthropic، و غیره) یا تمایل به استفاده از مدل های محلی رایگان

## دستورالعمل نصب دقیق

### برای کاربران Docker
1. [Docker Desktop] (https://docker.com/products/docker-desktop) را نصب کنید
2. نصب [Docker Compose] (docker-compose.md) را دنبال کنید
3. راهنمای گام به گام را دنبال کنید
4. دسترسی به «http://localhost:8502».

### برای نصب منبع (توسعه دهندگان)
1. Python 3.11+، Node.js 18+، Git را نصب کنید
2. [از منبع] (from-source.md) را دنبال کنید
3. «make start-all» را اجرا کنید
4. دسترسی به «http://localhost:8502» (frontend) یا «http://localhost:5055» (API)

## بعد از نصب

زمانی که راه افتادید:

1. **پیکربندی مدل ها** - ارائه دهنده هوش مصنوعی خود را در تنظیمات انتخاب کنید
2. **اولین دفترچه یادداشت بسازید** - سازماندهی تحقیق را شروع کنید
3. **اضافه کردن منابع ** - PDF، لینک های وب، اسناد
4. **کاوش ویژگی** - چت، جستجو، تحولات
5. **راهنمای کامل را بخوانید** - [راهنمای کاربر](../3-USER-GUIDE/index.md)
