# واژه‌نامه فارسی محصول

این واژه‌نامه برای یکدست‌سازی متن رابط کاربری فارسی (fa-IR) تهیه شده است.

## اصطلاحات اصلی

| English | واژه مصوب فارسی | توضیح کاربرد |
|---|---|---|
| Source | منبع | هر فایل/لینک/متن واردشده به سیستم |
| Notebook | دفتر | فضای کاری شامل منابع و یادداشت‌ها |
| Note | یادداشت | متن کاربر داخل دفتر |
| Insight | بینش | خلاصه/برداشت خودکار از محتوا |
| Ask | پرسش | حالت پرسیدن سؤال از پایگاه دانش |
| Search | جستجو | حالت جستجوی مستقیم در محتوا |
| Transformation | تبدیل | تبدیل محتوا با پرامپت یا قالب |
| Model | مدل | مدل هوش مصنوعی |
| Embedding | امبدینگ | نمایش برداری متن برای بازیابی |
| Context | زمینه | محتوای ارسالی همراه درخواست |
| Credential | اعتبارنامه | اطلاعات اتصال به ارائه‌دهنده مدل |
| API Key | کلید API | کلید دسترسی به سرویس |
| Delete | حذف | حذف مورد انتخابی |
| Save | ذخیره | ثبت تغییرات |
| Retry | تلاش مجدد | اجرای دوباره عملیات شکست‌خورده |

## اصول لحن

- لحن: محترمانه، ساده و مستقیم.
- CTAها: فعلی و روشن باشند (مثل «ذخیره تغییرات»، «حذف دائمی»).
- پیام خطا: کوتاه، روشن و قابل اقدام.
- پیام‌های دسترس‌پذیری (aria/alt/title): توصیفی و کاربرمحور باشند.

## الگوهای پیشنهادی

- خطای عمومی: «خطای غیرمنتظره‌ای رخ داد»
- خطای بارگذاری: «بارگذاری {مورد} ناموفق بود»
- موفقیت: «{مورد} با موفقیت ذخیره شد»
- حذف: «آیا از حذف این {مورد} مطمئن هستید؟»
